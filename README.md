# PHPアンケート集計アプリ

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - アンケートに答えるとローカルのテキストファイルにデータが溜まり、集計結果が見れるアプリです。
  - ジーズアカデミーのPHP課題で作りました。

## 工夫した点

  - アンケートフォームはCSSでスタイリング
  - Chart.jsでグラフを作成

## 苦戦した点

  - JavascriptとPHPの使い分けがよくわからず、結局、サーバーとの通信はPHP、それ以外はJavascriptでやるようにしてみました。

## 参考にした web サイトなど
